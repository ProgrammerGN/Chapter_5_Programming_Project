Chapter 5 Programming Project

Hello Professor! Here is my submission for the programming project for chapter 5. Below is a list of the five specific projects as well as a description for them. I also included the two data files (LineUp.txt and People.txt) since two of the programs read from them.

Population.cpp: Asks for a starting population, an average daily increase, and a number of days, then uses a loop to display the population for each day.
Student_Lineup.cpp: Reads the student names from LineUp.txt and reports how many students there are, who is at the front of the line, and who is at the end.
Population_Bar_Chart.cpp: Asks for a data file and a town name, then displays a bar chart of the town's population growth with one asterisk per 1,000 people.
Patterns.cpp: Uses nested loops to display two patterns of plus signs, one that grows from 1 to 10 and one that shrinks from 10 back to 1.
Guessing_Game.cpp: Generates a random number and lets the user keep guessing until they get it, telling them if each guess is too high or too low, then reports the number of guesses.
